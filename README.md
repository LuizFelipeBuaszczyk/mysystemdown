# MySystemDown
Este repositório é uma API para o um projeto completo do sistema ***MySystemDown***. O sistema é uma solução centralizada para verificar a saúde de diversos serviços locais. 
O usuário cria *systems* que possuem vários *services* no qual serão consumidos por um bot previamente configurado em cada sistema.  

## Tecnologias
- Python
- Django Rest Framework
